# Clon de Netflix - Proyecto Colaborativo

## Descripción del Proyecto
Este proyecto es un clon visual estático de la plataforma de streaming **Netflix**. Fue desarrollado de manera colaborativa para poner en práctica la maquetación responsiva y el trabajo en equipo mediante control de versiones.

## Componentes de Bootstrap Utilizados
Para este desarrollo, hicimos uso intensivo del framework Bootstrap 5. Entre los componentes principales se encuentran:
* **Grid System:** Para alinear las tarjetas de películas y lograr el diseño responsivo en diferentes pantallas (`row`, `col`).
* **Cards:** Para maquetar las portadas de las películas de forma uniforme (`card`, `card-body`, `card-img-top`).
* **Buttons:** Para los llamados a la acción de "Ver más" (`btn`, `btn-danger`).
* **Navbar:** Para la navegación superior responsiva de la plataforma.
